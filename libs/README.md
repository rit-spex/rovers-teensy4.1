# Libraries

These are files that need to be shared between the different subsystems to work with different peripherals.

Note for library names we are using upper camel case

CAN - library to allow communication over CANbus using Tx Rx Pins

SMC - library for communication for SMC motors on the arm using I2C

SPARKMAX - OLD Libraray that was meant to control the sparkmax, It was abandanded in the end due to migration to ODrives

TICT249 - library that is potentially abandanded, files are most empty but it does mention I2C

Version - library for CI/CD to keep track of the software versions. This is sources from https://github.com/Strooom/demoCloudBuilds/tree/main, NOTE this is copied and IS NOT MY WORK!!!
